# Week9-Classes
Create one Python file homework9_firstname_lastname.py and perform the following exercise:
## Python Classes Exercise: Medical Record System
Students will learn to use Python classes by creating a simple medical record system. They will define a class, create objects, and use methods to manipulate patient data.

## Instructions:
### Create a Patient class with the following attributes:

1) patient_id (integer)
2) name (string)
3) age (integer)
4) diagnosis (string)
5) charges (float)

### Implement Methods:

- display_info() → Prints the patient’s details.
- update_diagnosis(new_diagnosis) → Updates the patient's diagnosis.
- add_charges(amount) → Adds an amount to the patient's charges.

Create at least three patient objects with sample data.

### Call the methods to:

Display patient information.
Update a patient’s diagnosis.
Add charges to a patient's bill.

Bonus: Store multiple patients in a list and print all records.
